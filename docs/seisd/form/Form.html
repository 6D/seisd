Constructor Detail
Form()Constructor
public function Form(urlPHP:String)

Cria um formulário e permite as funções mais comuns

Parameters
urlPHP:String — Url de destino ao enviar. Atenção aos textfield.name pois serão os valores recebidos pelo $_POST do php. Ex: textfield.name = 'mensagem' resultará em $_POST['mensagem'] O php deve retornar "success=mail()" para ele poder validar a resposta do servidor. Caso contrário, sempre interpretará como enviado com sucesso.

public function add(input:*, type:String = null, required:Boolean = true):void

Adiciona um textfield ao form no formato de FormItem O texto padrão será pego pelo textfield.text. A quantidade de caracteres será pego pelo textfield.maxChars

Parameters

input:* — Textfield, Combox (evoluir mais...)
 
type:String (default = null) — Tipo do campo, consulte a classe FormItemType para detalhes. O valor null permite a inserção de qualquer caracter
 
required:Boolean (default = true) — Obrigatoriedade do campo ao enviar
